# LokaFood 🍜

### Platform Wisata Kuliner Nusantara

---

## Cara Menjalankan

1. Pastikan Flutter sudah terinstall
2. Clone atau extract project ini
3. Jalankan perintah berikut di terminal:

```bash
flutter pub get
flutter run -d chrome
```

---

## Struktur Proyek

```
lib/
├── main.dart                    # Entry point aplikasi
├── models/
│   └── kuliner_model.dart       # Model data kuliner
├── data/
│   └── kuliner_data.dart        # Data kuliner & rute
└── screens/
    ├── splash_screen.dart       # Halaman splash (Stateful)
    ├── onboarding_screen.dart   # Halaman onboarding (Stateful)
    ├── home_screen.dart         # Halaman utama (Stateless)
    ├── explore_screen.dart      # Halaman explore (Stateful)
    ├── detail_screen.dart       # Halaman detail (Stateless)
    ├── route_screen.dart        # Halaman rute (Stateful)
    └── profile_screen.dart      # Halaman profil (Stateful)
```
